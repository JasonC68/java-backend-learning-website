# 打卡表 · 复习次数 ＋/－ 按钮设置说明

> 前提：Excel 的按钮靠「宏(VBA)」驱动，文件必须保存为 **启用宏的工作簿 `.xlsm`**。
> 列约定：F列 = 复习次数，G列 = 上次复习日期（代码会自动填今天）。

---

## 方案一（推荐）：双击单元格 +1，最省事，不用画按钮

不需要任何按钮，双击「复习次数」单元格就 +1，并自动把当天日期填进「上次复习日期」。

步骤：

1. 把文件**另存为**：类型选「Excel 启用宏的工作簿 (*.xlsm)」。
2. 按 `Alt + F11` 打开 VBA 编辑器（Mac：`Option + F11`）。
3. 左侧双击「打卡表」这个 Sheet 对应的对象（不是 Module），把下面代码粘进去：

```vba
Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)
    If Target.Row >= 2 And Target.Column = 6 Then       ' 双击 F列(复习次数)
        Target.Value = Target.Value + 1
        Cells(Target.Row, 7).Value = Date               ' 自动填上次复习日期
        Cancel = True                                   ' 阻止进入编辑状态
    End If
End Sub
```

4. 保存。回到表格，双击任意一行的「复习次数」单元格即可 +1。
   （想减一可以手动改，或配合方案二的按钮。）

---

## 方案二：两个按钮（＋1 / －1），作用于当前选中行

1. 同样先**另存为 `.xlsm`**。
2. `Alt + F11` → 菜单 `插入 > 模块`，粘贴：

```vba
Sub 复习加一()
    Dim r As Long: r = ActiveCell.Row
    If r < 2 Then Exit Sub
    With Worksheets("打卡表")
        .Cells(r, 6).Value = .Cells(r, 6).Value + 1
        .Cells(r, 7).Value = Date
    End With
End Sub

Sub 复习减一()
    Dim r As Long: r = ActiveCell.Row
    If r < 2 Then Exit Sub
    With Worksheets("打卡表")
        If .Cells(r, 6).Value > 0 Then .Cells(r, 6).Value = .Cells(r, 6).Value - 1
    End With
End Sub
```

3. 开启「开发工具」选项卡：
   - **Windows**：文件 > 选项 > 自定义功能区 > 右侧勾选「开发工具」。
   - **Mac**：Excel > 偏好设置 > 功能区和工具栏 > 勾选「开发工具」。
4. `开发工具 > 插入 > 按钮(窗体控件)`，在表格上画一个按钮，弹窗里指定宏 `复习加一`；再画一个指定 `复习减一`。可右键按钮改文字为「＋」「－」。
5. 用法：先用鼠标点中某道题所在的任意单元格，再点按钮，对应行的复习次数就会变化。

---

## 常见问题

- **打开时提示"宏已被禁用"**：点黄条上的「启用内容」即可。
- **不想每次确认安全性**：文件 > 选项 > 信任中心 > 信任中心设置 > 宏设置，可调整（注意只对你信任的文件这么做）。
- **Mac 上画按钮麻烦**：直接用方案一的双击方式，体验更顺。
- **务必存成 .xlsm**：存回 .xlsx 会丢掉所有宏。
